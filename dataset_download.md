# Dataset AvailabilityThe AgroNLP-SimAssist corpus consists of- PlantVillage-derived disease labels- Symptom descriptions- Agricultural knowledge sources- Farmer QA repositories## Dataset Contents- 3,200 multilingual agricultural queries- Disease annotations- Advisory annotations- BIO entity labels## ConstructionInitial textual queries were generated using a large language model conditioned on disease labels and symptom descriptions.All generated queries underwent manual review and correction by agricultural domain experts before inclusion in the final corpus.Inter-annotator agreementCohen's Kappa = 0.89## Data SplitTraining70%Validation15%Testing15%Exact split indices are provided in```train_validation_test_split.csv```
